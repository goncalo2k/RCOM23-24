# Project 2

## Download Application

### How to run

<code>
$ make clean

$ make run1
</code>

or

<code>
$ make clean

$ make run2
</code>

or

<code>
$ make clean

$ make run3
</code>

<code>
$ make clean

$ ./download ftp://[<user>:<password>@]<host>/<url-path></code>

Example:

<code>$ ./download ftp://ftp.up.pt/pub/kodi/robots.txt
</code>
